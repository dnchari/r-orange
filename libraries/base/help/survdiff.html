Introduction

Performs survival difference analysis. This is similar to the Cox widget.

Details

This widget compares groupings of survival tables.  As with the Cox widget the user must enter time and outcome data using the formula entry commands.  These are populated on widget connect.

The value rho can be set between 0 and 1 with 0 resulting in the log-rank test and 1 resulting in the Peto and Peto modification of the Gehan-Wilcoxon test.

Outputs of the comparison are shown in the RoutputWindow.
