Introduction

Plots a survival object, usually from Survival Fit

Details

Results from comparing survival objects can be plotted using the Kaplan-Meier method.  This widget generates Kaplan-Meier plots from data fits generated using the Survival Fit Widget.  Plotting occurs on widget connect.

Plotting options in the widget dialog allow the user to set the main title as well as axis titles.  The 'X Scale Adjust' option can be used to adjust the X scale, for example, changing from months to days or years.

The size of the text can be adjusted using the 'Text Magnification Percent' line edit.
